# Variable Editor

A SillyTavern extension for viewing and editing local and global variables.

## Installation

Use ST's inbuilt extension installer with this URL:  
[https://github.com/Galfurian/st-variable-editor/](https://github.com/Galfurian/st-variable-editor/)

## Usage

- Slash command `/variableeditor` to toggle the panel.
- The panel displays local (chat-specific) and global variables in JSON format.

## Requirements

SillyTavern version >=1.10.10
